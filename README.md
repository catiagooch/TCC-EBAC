# TCC-EBAC

# UI Automation Project with Cypress

This project is a simple UI automation suite built using Cypress.  
The goal is to validate the functionality of adding different products to the shopping cart.

# Feature

- Add 3 different products to the cart
- Validate successful addition of each item
- Execution through terminal commands

# Tools

- Cypress for end-to-end testing
- Fixtures for test data
- Page Object Model for organized and maintainable code
- Custom Commands to reuse logic across tests
